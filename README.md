# Инструкция по начальной настройке
Шаг 1
Первоначально создайте группу и бота в Телеграмм. Они нужны для отправки уведомлений специалистам, о  созданных или измененных документах «Обслуживание клиента». Уведомления формируются при создании нового документа и при изменении даты и времени выполнения работ и ФИО специалиста, в уже существующем документе. 
Так же необходимо заполнить константы « ВКМ_ТокенУправленияТелеграмБотом» и «ВКМ_ИдентификаторГруппыДляОповещения».

Шаг 2 
Заполните регистр сведений «Условия оплаты сотрудников», укажите оклады у всех сотрудников и процент от продаж (если процент нулевой, то необходимо указать значение 0, не оставлять пустым).

Шаг 3
Заполните константу «НДФЛ (ВКМ)», актуальной процентной ставкой НДФЛ.

Шаг 4
В справочник «Графики работы» заведите необходимый вид графика работы, например пятидневка. Затем в подсистеме «Сервис» выберите обработку «Заполнение графика работы». В поле «График работы», выберите нужный график работы из справочника «Графики работы», созданный ранее. В поле «Выбор периода» выберите период (год) для заполнения графика. В поле «Выходные дни», через  запятую (без пробелов), перечислите порядковые номера выходных дней недели. Например: 6,7 (как суббота, воскресенье). Нажмите кнопку «Заполнить».
Далее перейдите в регистр «График работы» и проверьте корректность заполнения графика.

Шаг 5
В справочнике «Пользователи» создайте для каждого сотрудника пользователя. Например: Бухгалтер Иванова И.И., как пользователь «Иванова». На вкладке «Права доступа» назначьте каждому пользователю, соответствующий должности, профиль группы доступа. Наименования профиля группы доступа, созданы для Вашего удобства интуитивно понятными.
Пример: для специалистов профиль группы доступа - «Специалист», для кадровика – «Кадровик расчетчик». Все созданные профили групп доступа являются поставляемыми и не могут быть изменены пользователем самостоятельно. Подробнее с составом ролей можно ознакомиться по следующему пути: «Администрирование» - «Настройки пользователей и прав» - «Профили групп доступа» - папка «ВКМ»


Шаг 6
Заполните константы: «Номенклатура абонентская плата(ВКМ)» и «Номенклатуры работы специалиста(ВКМ)». Они заполняются из справочника «Номенклатура» соответствующим по смыслу наименованием, при необходимости создайте нужную номенклатуру.

Шаг 7
1.	Проинструктируйте кадровика о необходимости создать в справочнике «Сотрудники» ФИО всех сотрудников и указать для каждого одну из двух категорий «Административный персонал» или «Специалист» (необходимо для расчета Фиксированной премии и использования планировщика «Календарь специалистов (ВКМ)»).

В документе Начисление фиксированной премии(ВКМ), есть возможность автоматически рассчитать премии сотрудников, относящихся к категории «Специалист». Для этого нужно на жать на кнопку «Премия специалистов заполнить». Для сотрудников категории «Административный персонал», премия вносится в ручную, в этом же документе.

В документе «График отпусков (ВКМ)», после внесения данных, можно проверить превышение запланированных отпусков, нажав на кнопку «Проверить превышение отпуска» (будут подсвечены сотрудники, у которых общее количество запланированных дней отпуска превышает 28 дней). Так же можно на графике, проверить пересечение запланированных отпусков, для этого необходимо нажать на кнопку «Открыть график», а в открывшемся окне нажать кнопку «Заполнить».

2.	Проинструктируйте бухгалтера о возможности автоматического формирования документов по реализации за месяц с помощью обработки «Массовое создание актов (ВКМ)». Период в обработке должен быть выбран - предыдущий месяц. 
При создании реализации в ручную, для заполнения табличной части, воспользуйтесь кнопкой «Заполнить (ВКМ)». В табличную часть попадет сумма абонентской платы за месяц и сумма выполненных работ специалиста, за месяц (если работы не выполнялись, данная строка не заполнится). Период будет выбран, начало и конец месяца, от даты документа. 

При не заполненных константах: Номенклатура абонентская плата(ВКМ) и  «Номенклатуры работы специалиста(ВКМ)», табличная часть документа, заполнена не будет.

3.	Проинструктируйте менеджера, что в справочнике «Договоры контрагента» добавился вид договора «Абонентское обслуживание(ВКМ)». Для клиентов с кем подписаны договоры на абонентское обслуживание, необходимо создать,  отдельный договор. В поле «Вид договора» выбрать «Абонентское обслуживание(ВКМ)»,  в полях «Начало действия договора(ВКМ)» и Окончание действия договора (ВКМ), указать начало и окончание действия договора, соответственно. Внести стоимость абонентской платы, в поле «Сумма абонентской платы (ВКМ)». Внести стоимость, одного часа  работы специалиста, в поле «Стоимость работы за час (ВКМ)».
А так же о возможности планировать работу специалистов через планировщик «Календарь специалиста». В нем можно создать новый документ «Обслуживание клиента (ВКМ)» или отредактировать уже существующий. В нем более наглядно видны запланированные обслуживания, по каждому специалисту, на определенную дату.
