# Инструкция запуска сценария Проверка работы блока Обслуживание клиентов.
## Перед запуском сценария необходимо подготовить Базу и Vanessa Automation  для запуска сценария.
1.	Заполнить справочник Организации. Создать одну организацию с наименованием:
Основная организация

2.	Заполнить справочник Сотрудники. Создать трех сотрудников с именами:
Специалист Иванов
Специалист Петров
Специалист Сидоров

3.	Заполнить справочник Контрагенты. Создать три компании с наименованиями
Кактус
Репей
Ромашка

5.	Заполнить справочник Договоры. Создать три договора с наименованиями:
Договор Кактус
Договор Репей
Договор Ромашка
Вид договора – Абонентское обслуживание.
Действие договора - (произвольные данные, но на момент создания документов, договор должен быть действующим)
Сумма абонентской платы – 10 000
Стоимость часа работы специалиста – 5 000

6.	Заполнить константы.
ВКМ_НоменклатураАбоненскаяПлата – НоменклатураАбоненскаяПлата
ВКМ_НоменклатурыРаботыСпециалиста – НоменклатурыРаботыСпециалиста

7.	Заполнить Регистр ВКМ_УсловияОплатыСотрудников (произвольные данные).

8.	Удалить все созданные документы ВКМ_ОбслуживаниеКлиентов и РеализацияТоваровУслуг

9.	В Vanessa Automation добавить трех  Клиентов тестирования

Клиент Менеджер – Доп параметр /N"Менеджер"
Клиент Специалист – Доп параметр /N"Специалист"
Клиент Бухгалтер – Доп параметр /N"Бухгалтер"
